# Frontend (Vite) — README

Ce dépôt contient l'interface front construite avec Vite.

## Prérequis
- Node.js (>=16 recommandé)
- npm, yarn ou pnpm

## Installation
1. Ouvrir le dossier :
  cd /front
2. Installer les dépendances :
  - pnpm: `pnpm install`
  
## Démarrage
  Lancer le serveur de développement (exécuter plusieurs fois si besoin) :

  ```bash
  pnpm run dev
  ```
